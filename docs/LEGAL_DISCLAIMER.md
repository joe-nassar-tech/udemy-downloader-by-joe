# Please Use This Kindly and Legally

This tool is for your own learning materials when you have permission to save them.

## Good uses

- Saving your own allowed course files.
- Keeping a personal backup where the provider allows it.
- Using the provider's official offline feature.

## Not okay

- Sharing downloaded paid lessons.
- Sharing `cookies.txt`, `cookies.json`, passwords, or keys.
- Trying to break, guess, extract, or bypass DRM.
- Using someone else's account or cookies.

DRM is a provider lock. Some videos can have different authorization. If a protected lesson fails, ask the provider or instructor for a supported way to watch it offline.

You are responsible for following the course provider's terms, copyright law, and local rules.
